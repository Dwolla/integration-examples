# Environment variables

Copy these into a local `.env.local` file when running the sample.

```
DWOLLA_ENV=sandbox
DWOLLA_KEY=your_dwolla_key
DWOLLA_SECRET=your_dwolla_secret

# Checkout.com
CKO_PUBLIC_KEY=pk_sbox_your_public_key
CKO_SECRET_KEY=sk_sbox_your_secret_key
CKO_ENV=sandbox

# App base URL used in redirects/callbacks
NEXT_PUBLIC_APP_BASE_URL=http://localhost:3000
```
